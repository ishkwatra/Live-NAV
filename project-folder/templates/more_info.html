How It Works

This tool estimates the current NAV of mutual funds by using portfolio holdings, their weightages, and live stock prices. The calculation follows these steps:

  1. Fetch the previous day's NAV from trusted sources like Moneycontrol.
  2. Retrieve current and previous stock prices from Yahoo Finance.
  3. Estimate percentage changes for each stock and multiply them by their respective weights in the portfolio.
  4. Sum up these weighted changes to estimate the overall NAV change.
  5. Provide a detailed report, including profits and errors for stocks with missing data.

Current Features

  • Support for 5 mutual funds with detailed calculations.
  • Real-time fetching of stock prices and NAV data.
  • Error handling for unavailable stock data.
  • Profit summary based on user-provided investment data.
  • Expandable detailed report for each mutual fund.

Future Enhancements

  • Multi-threading to speed up stock price fetching.
  • Support for custom portfolios uploaded via CSV files.

  • Historical NAV trends and performance comparison between funds.
  • A user-friendly interface for easier access and analysis.
